# Weather Website

This project is a simple weather website created with HTML and CSS.

## Features
- Shows today's weather
- Shows tomorrow's forecast
- Includes humidity and wind speed
- Uses styling for a cleaner layout
- Includes a link to a weather website for more information

## Files
- `index.html`: main webpage
- `README.md`: project description
